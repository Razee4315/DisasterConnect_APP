# Code of Conduct

## Our Standards

This project is committed to providing a welcoming and respectful environment for everyone. We expect all participants to:

- Be respectful and considerate in all interactions
- Accept constructive feedback gracefully
- Focus on what is best for the community and the project
- Show empathy towards other community members

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, intimidation, or discrimination of any kind
- Personal attacks or derogatory comments
- Publishing others' private information without consent
- Any conduct that would be considered inappropriate in a professional setting

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and any other communication channels associated with this project.

## Enforcement

Instances of unacceptable behavior may be reported by contacting the project maintainer at [GitHub Issues](https://github.com/Razee4315/DisasterConnect_APP/issues). All reports will be reviewed and addressed appropriately.

Maintainers have the right to remove, edit, or reject contributions that do not align with this Code of Conduct, and may temporarily or permanently ban any contributor for behavior deemed harmful.
